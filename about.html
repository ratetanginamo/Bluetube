About BlueTube

BlueTube is a student-developed web application built by IT students from Surigao Education Center (SEC), Surigao City, Philippines. Powered by the YouTube Data API v3 and hosted on GitHub Pages, it delivers trending videos and instant search — all wrapped in a neon-cyberpunk interface.

Our Mission

To empower SEC IT students with a fast, beautiful, and accessible platform for discovering high-quality programming tutorials, web development guides, and tech content — all in one place.
